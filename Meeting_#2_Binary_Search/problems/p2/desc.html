Problem Statement

Given an array of $N$ ($1 \le N \le 10^5$) unique, sorted integers, please write a program to answer $K$ ($1 \le K \le 5 \times 10^5$) queries.
In each query, you will be given two integers $a$ and $b$. Please write a program to calculate the number of integers that are within $[a, b]$.

Input Specifications

On the first line are two integers: $N$ and $K$, representing the number of elements in the array and the number of queries. On the following $N$ lines are $N$ sorted integers. On the following $K$ lines are two integers $a_i$ and $b_i$, indicating the $i$'th query. It is guaranteed that $a_i \le b_i$.

Output Specification

For each query, output the number of elements in the range of $[a_i, b_i]$.

Sample Input

5 2
1 2 4 5 6
1 5
4 8

Sample Output

4
3

Explaination of Sample Input / Output

For the first query, there are 4 integers within the range of $[1, 5]$. They are $1$, $2$, $4$, and $5$.
For the second query, there are 3 integers within the range of $[4, 8]$. They are $4$, $5$, and $6$.
